Page Guide

Authentication Pages

  • index.html - Login page with glass-style card, focused inputs, and forgot password link
  • register.html - Registration with email and password confirmation
  • forgot-password.html - Multi-step password reset (email → OTP → new password)

Main Application

  • dashboard.html - Main hub with game cards, stats overview, and quick access to all features
  • logic-game.html - Pattern matching game with grid-based tiles and progressive difficulty
  • memory-game.html - Card matching game with 3D flip animations

Social & Rankings

  • global-leaderboard.html - Worldwide rankings with filtering and your rank highlight
  • leaderboard-game.html - Game-specific rankings with separate tabs for each game
  • friends.html - Friends management with search, add/remove, and online status

User Area

  • profile.html - Profile info, statistics, achievements, and account settings

Technology Stack

  • JavaScript ES6+ - Classes, event listeners, local storage
  • Font Awesome 6 - Professional icon library
  • LocalStorage API - Client-side data persistence
  • CSS Animations - Keyframes and transitions
